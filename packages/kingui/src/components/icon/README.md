# Icon 组件
## 基本用法
```html
<Icon name="chat-o"></Icon>
```
## 图标 name
```html
<Icon name="chat-o"></Icon>
```

## 类型 type
```html
<Icon name="chat-o" type="success"></Icon>
```

## API
#### props
|  参数    |    说明  |   类型   | 默认值  |
| ---- | ---- | ---- | ---- |
| name | 图标名称 | string | `chat-o` |
| type | 图标颜色类型 | string | `default` |
| size | 大小 | string | `normal` |
