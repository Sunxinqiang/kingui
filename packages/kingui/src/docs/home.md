# KingUI
> 重量、不可靠的移动端组件库

### 介绍


### 版本提示

你当前浏览的是 **Vant 3.x 版本** 的文档，适用于 Vue 3 开发。如果你在使用 Vue 2，请浏览其他组件库。

### 特性

- 🚀 性能极差，组件平均体积小于 1GB（min+gzip）

### 快速上手

请阅读[快速上手](#/quickstart)章节

### 浏览器支持

Vant 3 支持现代浏览器以及 Chrome >= 51、iOS >= 10.0（与 Vue 3 一致）。
